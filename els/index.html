The Early Learning Suite, NetsourceELS, is a platform that assists school districts in the areas of licensing, professional development, quality and improvement for early childhood education.
Early Learning Suite formal release
NetsourceELS, is a customized version of NetsourceLX, a cloud-based knowledge and eLearning platform. NetsourceELS consists of various modules:

Licensing

This module provides a controlled workflow based platform for the licensing process for early childhood learning facilities. The system incorporates a rich database on various data for each facility. The platform is designed to reduce the level effort required during the licensing process while providing a platform for monitoring and tracking. (Currently being developed)

QRIS

The QRIS Module provides a customizable framework that enables school districts to establish a Quality Rating & Improvement System (QRIS) for early childhood education. This includes web capabilities to track, measure and capture various metrics, attendance and assessment data. The module provides the ability for the collaboration between school districts, oversight staff and facility personnel. (Currently being developed)

Professional Development

This module provides a platform for the tracking of professional development for early learning professionals. It also manages both classroom-based and online eLearning courses. This module leverages the core capabilities of the NetsourceLX product but is designed specifically for the early learning setting. (Development Completed)

Child Subsidy

This is a module than provides access to information in the other modules to support child subsidy programs. Development of this module has not started.

Customization & Integration

The strength of our platform is that it can be customized to meet the unique needs of each state and school district. Additionally, integration with on-premise existing systems can be leveraged using custom APIs.

Cloud-based

NetsourceLX can be implemented very quickly without the need to purchase hardware and add additional burden on your internal IT team.

It runs on a secure, scalable and highly available platform that can grow with your needs.
